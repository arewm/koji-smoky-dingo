Plugin Commands¶

Koji Smoky Dingo provides a number of additional commands to the koji CLI.

Informational¶

These read-only commands gather and display information from Koji, and do not require any special permissions to run.

  • koji affected-targets
  • koji cginfo
  • koji check-hosts
  • koji client-config
  • koji filter-builds
  • koji filter-tags
  • koji latest-archives
  • koji list-btypes
  • koji list-build-archives
  • koji list-cgs
  • koji list-component-builds
  • koji list-env-vars
  • koji list-rpm-macros
  • koji list-tag-extras
  • koji open
  • koji perminfo
  • koji userinfo

Administrative¶

These commands modify aspects of Koji and are subsequently gated behind some level of administrative permission.

See Koji's Permission System - Administration

  • koji block-env-var
  • koji block-rpm-macro
  • koji bulk-move-builds
  • koji bulk-tag-builds
  • koji bulk-untag-builds
  • koji remove-env-var
  • koji remove-rpm-macro
  • koji renum-tag-inheritance
  • koji set-env-var
  • koji set-rpm-macro
  • koji swap-tag-inheritance
